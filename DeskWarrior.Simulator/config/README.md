# Simulator Config (시뮬레이터 전용 설정)

## 목적
이 config 폴더는 **시뮬레이터 전용**입니다. 게임과 완전히 분리되어 있습니다.

## 구조 분리
```
/config/                        ← 게임 실제 사용 config
/DeskWarrior.Simulator/config/  ← 시뮬레이터 전용 config (이 폴더)
```

## 워크플로우
1. 시뮬레이터에서 밸런스 실험 수행
2. 이 폴더의 config 값을 자유롭게 수정
3. 밸런스 확정 후 `/config/`로 수동 반영
4. 게임은 `/config/`만 사용

## 주의사항
- 시뮬레이터 config 수정 = 게임에 영향 없음
- 밸런스 확정 전까지는 게임 config를 건드리지 않음
- 실험값과 게임값을 완전히 분리하여 안전하게 테스트 가능
