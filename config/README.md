# Game Config (게임 실제 사용 설정)

## 목적
이 config 폴더는 **게임에서 실제 사용**하는 설정입니다.

## 구조 분리
```
/config/                        ← 게임 실제 사용 config (이 폴더)
/DeskWarrior.Simulator/config/  ← 시뮬레이터 전용 config
```

## 수정 규칙
- **직접 수정 금지** - 시뮬레이터에서 밸런스 확정 후 반영
- 밸런스 미확정 값을 게임에 적용하면 플레이어에게 직접 영향

## 워크플로우
1. 시뮬레이터에서 밸런스 실험 및 확정
2. 확정된 값을 이 폴더로 수동 복사
3. 게임 빌드 및 배포

## 주의사항
- 이 폴더 수정 = 게임 즉시 영향
- 실험값은 `/DeskWarrior.Simulator/config/`에서만 수정
- 밸런스 확정 전까지는 건드리지 않음
